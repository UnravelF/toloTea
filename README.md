Tolo奶茶是一个前端移动端的奶茶点餐系统，基于vue2与mockjs开发的移动端项目。基于mockjs产生模拟数据，编写后台模拟接口，通过拦截前端的网络请求传递模拟数据进行渲染页面，已经实现奶茶点餐系统的相关功能，内含多类功能化组件的封装运用。

### 功能简介

------

- 路由跳转
- 奶茶点餐
- 结算记录
- 订单获取

### 安装步骤

------

```bash
# 安装依赖
npm install

# 访问地址 http://localhost:8080
npm run serve

# 编译打包
npm run build
```
